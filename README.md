# um dashboard moderno de previsão do tempo construído com a API do HG Brasil

## Features

* Busca dados da API HG Brasil sobre a previsão do tempo em tempo real
* Possível ver a previsão de hoje e amanhã 
* Desenvolvido para funcionar em dispositivos mobile


## Tecnologias

* React (Vite)
* Tailwind CSS
* Axios

## Como rodar o código

* git clone https://github.com/Marcos5815/Previsao-do-tempo.git
* npm install
* npm run dev
